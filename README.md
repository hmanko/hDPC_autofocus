# hDPC_autofocus

This is repository containing the code, GUI and other required files for autofocusing procedure described in article "Nanoscale drift-correction in high-resolution optical microscopy inside thick biological tissues based on homogenized phase gradient imaging" which was submitted for publication.

User manual describes the procedure of installation, devices configuration and control of the autofocusing system. Two separate forlders contain GUI file generated by Qt5-tools designer and the main script for application with trans- and back-illumination modalities. 

Upcoming updates will include direct Python control.
